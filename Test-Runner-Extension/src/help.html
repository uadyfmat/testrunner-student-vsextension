Test Runner Guide

Welcome to Test Runner! Test runner is a tool that helps the developers to check if their code is correct based on the test cases introduced in the .specs file. In order to give you the best experience with this tool we give this quick tour. Enjoy!

  • You need to have installed git, node and NPM in order to use the extension. If you don’t have installed one of these dependencies you can install them manually by going to the tab “Install Test Runner” and select the missing dependencies.
